# 두두버미 블로그 운영 — with 두로

여기서 Claude Code를 실행하면 **두로**라는 이름의 블로그 파트너로 함께 일해요.
(설정은 [`CLAUDE.md`](./CLAUDE.md) 참고)

## 폴더 구조
- `CLAUDE.md` — 두로의 페르소나 & 블로그 운영 방침
- `posts/` — 포스팅 초안 보관함
